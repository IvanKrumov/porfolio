My portfolio page

Hi,

My name is Ivan and this is my portfolio web site. The purpose of it is to upload my CV and my certificates in environment created all by me. The site is hosted on bare metal machine located in my living room runing Ubuntu 20.04 LTS Server. I use Nginx as reverse proxy and Nginx Docker container to host the site. For storage I am using two drive rack mounted on the CD-ROM slot as both drives are set to work as RAID 1. As a backup machine I am using Raspberry Pi running Raspberry Pi OS(old Raspbian). For my home network I am using Archer C7 wireless rooter as I removed the stock software and installed openWRT.

I would like to thank you for the time you spent cheking my page. Please stay tuned for my new accoplishments and new improvements on my page.
