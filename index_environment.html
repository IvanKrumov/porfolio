This is my pipeline and works as it follows:
- Once the code is tested localy on my laptop(1) I push it in the GihHub repo(3).
- In the repo there is webhook set, that notifies the Jenkins which is installedd
on the Raspbery Pi(2) and Jenkins job deploys the site in the Virtual server(Apache2)
- If I am happy with the result I start bash script on the production server(4) and
code is finally pulled there.
