This is my pipeline and works as it follows:
Once the code is tested localy on my laptop I push it in the Github repo.
In the repo there is webhook set, that notifies the Jenkins which is installed
on the Raspbery Pi and Jenkins job deploys the site in the Virtual server(Apache2) on the Pi
If I am happy with the result I start bash script on the production server and
code is finally pulled there. The production server itself has Nginx reverse proxy with
Let's encrypt SSL sertificate and behind it there is Nginx Docker container that hosts the site.
